Algoritmo de Euclides: 
Este projeto aprensenta dois tipos de funções que executam de forma eficiente o algoritmo de Euclides, cuja função é encontrar o MDC(máximo divisor comum) entre dois números.

🛠️ Tecnologias e Conceitos: 
Python 3.x e Lógica Matemática - Algoritmo de Euclides📋

Pré-requisitos:
Python instalado;
Git;
Gerenciador de pacotes (pip);
